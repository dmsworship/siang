Ping Test Bot

Install

pkg update && pkg upgrade
pkg install nodejs git
npm install @whiskeysockets/baileys pino

---

Jalankan Bot

node index.js

Masukkan nomor:

628xxxxxxxxxx

Masukkan pairing code ke WhatsApp:

- Buka WhatsApp
- Linked Devices
- Link with code

---

Perintah Bot

Kirim pesan ke nomor WhatsApp bot (nomor yang kamu login di bot ini):

!ping_test

---

Catatan

- Bot harus sukses mengirim pesan ke semua nomor. Jika reconnect sebelum selesai berarti terkena batasan akun whatsapp kalian, artinya kurang badak hehehe!

Bot akan otomatis kirim pesan "P" ke semua nomor tersebut.
